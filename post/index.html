Hugo 始めました

github.io用にoctpressで少しだけ書いておりました。
このたび新しくPC環境を作り、BeagleBoneBlackも手元に来たことだしと
いろいろ手を加えたところ、わけわからんテンプレートを見事に破壊してしまい
ロールバックしないと記事の投稿も無視されるようになってしまいました。

そんなに書き溜めている分もないので、
巨大すぎて何してるのかよくわからん環境を放置して Hugo に移ってきました。
golang 大好きとかもありますけどこう、
ソフトウェアが何やってるのかはっきりわかるのはとても良いように思います。
見よう見真似でいじくりまわすと見た目はともかく
いじくった場所が変わるので大変わかりやすいです。

$ hugo new post/mynewentry.md

とかやって新しいファイルをつくって

$ hugo  server --buildDrafts

とかするとhtmlが生成されるのでそれをそのままgitにpushするだけです。
octpressと比べるとデプロイのところが自動になってないのが面倒かもしれませんが
gitだしそのうちシェルスクリプトにでもしようかなと。

ぼちぼちいきますのでよろしくお願いいたします。
