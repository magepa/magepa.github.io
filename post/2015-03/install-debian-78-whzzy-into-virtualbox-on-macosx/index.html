MacOSX の VirtualBoxにDebian7.8(whzzy)をインストール

単なるインストール記録です。

VirtualBox

何はともあれ開発環境が必要なので、VirtualBoxを用意しました。
以前はヘタレてUbuntsuなどを入れていたのですが
13ぐらいでイライラが絶頂に達したので Debian/GNU Linuxを入れることに。

最近のVirtualBoxはインストールしたらほぼ何も設定しなくても
NAT でネットワーク繋がるので、設定したのはHDのサイズぐらいのものです。
すばらしい。Sunに感謝。太陽に感謝。

Debian/GNU Linux

google先生でdebian ダウンロード　検索！
とやってでてきた Debian “wheezy” インストールCD/DVD用イメージから

PC/サーバー用 netinstイメージ 64 bit(amd64)

となっていた debian-7.8.0-amd64-netinst.iso をダウンロードしました。

インストールも簡単になっていてひとまずは何も考えず
SSHだけいれたデスクトップをxfaceで入れました。
最小構成で入れるとビルドのときにあれもこれもなくて大変というヘタレな理由です。

というわけでインストールはあっさり終了。 だいたいほったらかしで2時間ぐらいでしたでしょうか。

最低限の設定

sudoの設定して、コンパイル二とにかく時間がかかるのでスクリーンセーバーを停止しておきました。 徐々に環境構築していきます。
